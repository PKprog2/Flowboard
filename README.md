## ğŸ“– How to Use

### Creating Boards
1. Click the **"+ New Board"** button in the tabs
2. Type your board name (spaces allowed!)
3. Press Enter to save

### Creating Lists
1. Click **"+ New List"** at the end of your lists
2. Type the list name
3. Press Enter to create

### Adding Tasks
1. Click **"Add Task"** in any list
2. Type task names and press Enter to quickly add multiple tasks
3. Press Escape when done

### Managing Tasks
- **Complete**: Click the circle icon to mark complete (moves to Done)
- **Delete**: Click the trash icon (can undo with Ctrl+Z)
- **Move**: Drag and drop tasks between lists
- **Reorder**: Drag lists to reorder (Done stays at the end)

### Keyboard Shortcuts
- `Ctrl+Z` / `Cmd+Z` - Undo last action
- `Enter` - Save/Create
- `Escape` - Cancel editing
